# FdF
This repo provides, as the name already says, a minishell.  

Tested on macOS Monterey.

## Installation

#### 1) Clone this repository 
```
git clone git@github.com:JL1709/FdF.git
```

#### 2) Execute Makfile
```
cd FdF
make
```

#### 3)  Run program, for example:
```
./fdf maps/42.fdf
```
